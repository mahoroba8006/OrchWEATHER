1. 収集する情報

Google アカウント情報

Firebase Authentication（Google OAuth 2.0）を通じてログインする際、以下の情報を取得します。

  • 氏名
  • メールアドレス
  • プロフィール画像 URL

これらはログインセッションの維持と画面表示にのみ使用します。

ユーザー設定

積算開始日・基準温度・表示設定等のアプリ内設定を Cloud Firestore に保存します。これらはアプリの動作に必要な最低限の情報のみです。

AIコメントのキャッシュ

AI（Google Gemini）が生成した気象コメントは、過剰な API 呼び出しを防ぐため Firestore に一時保存されます。コメントには個人を特定できる情報は含まれません。

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • Google OAuth によるログイン認証・セッション管理
  • ユーザー設定（積算開始日・基準温度等）の保存・復元
  • AIコメントの生成キャッシュ管理

収集した情報を広告配信・第三者への販売・マーケティング目的に使用することはありません。

3. データの保管場所

  • 認証情報：Firebase Authentication（Google LLC 管理）
  • ユーザー設定・AIコメントキャッシュ：Cloud Firestore（Google LLC 管理、東京リージョン）
  • 気象データ：本アプリのサーバーには保存されません。Open-Meteo API からリアルタイムに取得し、ブラウザのメモリ内にのみ一時的に保持されます。

本アプリが独自に管理するサーバー（自社データベース）にユーザーの個人情報を蓄積することはありません。

4. 第三者サービスの利用

本アプリは以下の第三者サービスを利用しています。

Google LLC（Firebase・Gemini）

  • Firebase Authentication による Google ログイン認証
  • Cloud Firestore によるユーザー設定・AIコメントキャッシュの保存
  • Gemini API による気象コメントの AI 生成（送信されるのは気象数値データのみ。個人情報は送信されません）
  • Google Analytics による利用状況の分析（詳細は「8. アクセス解析ツール」をご覧ください）

Google プライバシーポリシー →

Open-Meteo（open-meteo.com）

  • 過去の気象実績データの取得（気温・降水量・日射量・日照時間 等）
  • 10日間予報データの取得

送信される情報は位置情報（緯度・経度）と日付範囲のみです。個人を特定できる情報は送信されません。

Open-Meteo 利用規約 →

気象庁（jma.go.jp）

  • 注意報・警報のリアルタイム取得（気象庁 防災情報 JSON）
  • 送信される情報は都道府県コードのみです。個人を特定できる情報は送信されません。

気象庁 著作権・リンク →

Cloudflare, Inc.

  • 本アプリのホスティング（Cloudflare Pages）
  • Gemini API へのリクエスト中継（Cloudflare Workers）

Cloudflare プライバシーポリシー →

Tally Forms（株式会社 Tally）

  • お問い合わせフォームの提供および受付内容の中継
  • フォーム送信時、入力された氏名・メールアドレス・お問い合わせ内容が Tally のサーバーを経由して当方の運用メールへ転送されます

Tally プライバシーポリシー →

5. セキュリティ・データ保護

  • 通信の暗号化：すべての通信は HTTPS（TLS）で暗号化されます。
  • OAuth 2.0 認証：Google の OAuth 2.0 プロトコルを利用し、パスワードを本アプリが取り扱うことはありません。
  • Firestore セキュリティルール：ユーザー自身のデータにのみアクセスできるよう制限しています。
  • 最小データ収集：アプリの機能提供に必要な最低限の情報のみを収集します。

6. 利用者の権利

ユーザーはいつでも以下の操作が可能です。

  • アプリからのログアウト
  • Google アカウントの設定から本アプリへのアクセス許可を取り消す

アクセス許可の取り消しは、Google アカウントの権限設定から「Orch.Weather」を削除することで行えます。アクセス許可を取り消した場合、Firestore に保存された設定データへのアクセスができなくなります。

保存データの削除をご希望の場合は、アプリ内のお問い合わせ機能よりご連絡ください。

7. Cookie・ローカルストレージ

本アプリはログインセッションの維持のため Firebase が発行する Cookie を使用します。また、以下の情報をブラウザのローカルストレージに一時保存する場合があります。

  • 選択中の表示設定（表示期間・グラフ種別 等）

これらローカルストレージへ保存する情報は、アプリの動作に必要な最小限のものであり、広告目的には使用しません。利用状況の分析に用いる Cookie については「8. アクセス解析ツール」をご覧ください。

8. アクセス解析ツール

本アプリは、利用状況の把握とサービス改善のため、Google LLC が提供するアクセス解析ツール「Google Analytics（GA4）」を使用しています。

  • Google Analytics は Cookie を利用して、ページの閲覧状況や機能の利用状況などの匿名のトラフィックデータを収集します。
  • 収集されるデータは匿名であり、個人を特定するものではありません。氏名・メールアドレス等の個人情報が Google Analytics に送信されることはありません。
  • 収集した情報は、どの機能がどの程度利用されているかの分析、およびサービス改善の目的にのみ使用します。

ブラウザの設定で Cookie を無効にする、または Google アナリティクス オプトアウト アドオン を利用することで、データの収集を拒否できます。詳細は Google のポリシー をご確認ください。
