# アプリの使い方ページ 実装プラン

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** ヘッダーに「使い方」ボタンを追加し、初回ユーザーが各機能を把握できる縦スクロール型ガイドページを実装する

**Architecture:** `topTab` に `'help'` を追加し、既存の状態ベースナビゲーションを拡張する。`prevTopTab` ref で遷移元タブを保持し「← 戻る」で復帰できる。HelpPage は独立したコンポーネントとして `src/components/HelpPage.tsx` に実装する。

**Tech Stack:** React 19 + TypeScript + Vite。外部ライブラリ追加なし。スタイルは既存の `glass-panel` / `app-container` / `glass-table` CSS クラスを流用。

---

| ファイル | 変更種別 | 内容 |

| `src/App.tsx` | 修正 | topTab型・prevTopTab ref・ヘッダーボタン・HelpPage描画 |
