# 日別予報レイアウト redesign — 設計書

**日付:** 2026-06-13
**対象ファイル:** `src/components/weather/DailyForecast.tsx`

---

## 目的

「いまの空」日別予報の表示を整理し、分割日（最初の3日）と非分割日（4日目以降）の視覚的整合性を高める。

---

## 現状の課題

現在の行構成（7行）：

| 行 | 分割日 | 非分割日 |
|----|--------|---------|
| 1 | 日付のみ（colSpan=3） | 日付＋天気テキスト（1セル） |
| 2 | 午前/午後/夜間ラベル（3セル） | **空セル** |
| 3 | 天気テキスト（3セル） | **空セル** |
| 4 | 天気アイコン（3セル） | 天気アイコン（1セル） |
| 5 | 降水確率（3セル） | 降水確率（1セル） |
| 6 | 最高/最低気温（colSpan=3） | 最高/最低気温（1セル） |
| 7 | ミニチャート / ガントバー | ← 同左 |

問題点：
1. 日付行の高さが不均一（非分割日は日付＋天気テキストの2行分）
2. 非分割日に空行が2行あり視覚的に間延び
3. 天気テキストが分割日はRow3、非分割日はRow1に入っており行が揃わない

---

## Row 1 詳細設計

### 分割日（colSpan=3）

```
┌──────────────────────────────────┐
│ 今日 6/13(金)                     │  ← 日付ラベル（左寄せ）
│  午前  │    午後    │    夜間     │  ← flex 3等分
└────────┴────────────┴────────────┘
```

- `verticalAlign: 'top'`（**現行の `'middle'` から変更**）、`paddingTop: '0.75rem'`、`paddingLeft: '0.5rem'`
- 日付ラベルの直下に `display: flex` の3等分ラベル行を配置
- 各ラベル: `午前` / `午後` / `夜間`（時刻表記なし）
- ラベル間境界: 2番目・3番目のみ `borderLeft: '1px solid var(--card-border-sub)'`（1番目には付けない）
- フォントサイズ: `0.65rem`、色: `var(--text-tertiary)`、fontWeight: 600

### 非分割日（1セル）

- `verticalAlign: 'top'`（**現行の `'middle'` から変更**）、`paddingTop: '0.75rem'`
- 日付ラベルのみ（天気テキストは Row 2 へ移動）
- Row 1 の高さは分割日の「日付＋ラベル行」に合わせて決まるため、非分割日の日付は Row 1 上端に揃う

---

## Row 2 詳細設計

### 分割日（3セル）

- 現行の「天気テキスト行」と同じ内容・スタイル
- 各セル: `amWeatherCode` / `pmWeatherCode` / `nightWeatherCode`（または Short）の `codeToLabel()`

### 非分割日（1セル）

- **現行は空セル → `codeToLabel(day.weatherCode)` を表示（Row 1 から移動）**
- フォントサイズ: `0.806rem`、色: `var(--text-tertiary)`、fontWeight: 500
